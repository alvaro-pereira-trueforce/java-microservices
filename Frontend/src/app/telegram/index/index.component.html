How to install Telegram Channel Integration

Getting Started

To start talking with your costumers using Telegram Bots, install Telegram Channel app from Zendesk Marketplace. Then, create your bot using @BotFather - you will get a token.

Alternatively just use the token from your existing telegram bot just follow the steps in Zendesk Support.

BotFather, create a new Telegram bot:

BotFather is the one bot to rule them all. It will help you create new bots and change settings for existing ones.

  1. Click here to open the BotFather chat.
  2. Write and send /newbot command to create a new bot. The BotFather will ask you for a name and username, then generate an authorization token for your new bot.
  3. The name of your bot is displayed in contact details and elsewhere.
  4. The Username is a short name, to be used in mentions and telegram.me links. Usernames are 5-32 characters long and are case insensitive, but may only include Latin characters, numbers, and underscores. Your bot's username must end in ‘bot’, e.g. ‘tetris_bot’ or ‘TetrisBot’.
  5. The token is a string along the lines of 110201543: AAHdqTcvCH1vGWJxf SeofSAs0K5PALDsaw that is required to authorize the bot to use the integration with Zendesk Support.
  6. Note: If your existing token is compromised or you lost it for some reason, use the /token command to generate a new one.

In Zendesk:

  1. Go to your Zendesk admin page and look out for Channel Integrations button at the side bar.
  2. Click on Telegram Channel then click on Accounts tab, click on Add account button in order to configure the Telegram Bot Token.
  3. Set an Integration Name to organize your accounts.
  4. Paste your Telegram Bot Token into Telegram token field.
  5. Click on Submit.

The new account is ready to start receiving your customers messages.
