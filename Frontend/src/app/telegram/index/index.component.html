How to install "Telegram Channel"

Getting Started

To start talking with your costumers using Telegram Bots, install our "Telegram Channel" app from Zendesk Marketplace. Then, create your bot using @BotFather, after creation you will get a token.

Alternatively just use the token from your existing telegram bot just following the Zendesk Support steps.

BotFather, create a new Telegram bot:

BotFather is the one bot to rule them all. It will help you create new bots and change settings for existing ones.

  1. Click here to open the BotFather chat.
  2. Open your Telegram app via Telegram Desktop or Web, a chat with the BotFather should be open.
  3. Write and send /newbot command to create a new bot. The BotFather will ask you a name for your bot.
  4. Insert a name for your bot (ex. Acme Support), the BotFather should ask you for a username for your bot.
  5. Insert a username for your bot, it must end with `bot` (ex. acme_support_bot or AcmeSupportBot).
  6. A congratulations message should be displayed with the information of your new Bot including its token.
  7. Copy your new Bot token.

Notes:

  • Your Bot’s name is displayed in Contact details and elsewhere.
  • Your Username bot is a short name to be used in mentions and telegram.me links. Usernames are 5-32 characters long and are case sensitive, but may only include Latin chatacters, numbers, and underscores. Your Bot’s username must end in `bot`, ex. ‘tetris_bot’ or ‘TetrisBot’.
  • The token is a string along the lines of 110201543:AAHdqTcvCH1vGWJxfSeofSAs0K5PALDsaw that is required to authorize the bot to use the integration with Zendesk Support.
  • If your existing token is compromised or you lost it for some reason, use the /token command to generate a new one.

Zendesk Support Steps:

  1. Go to your Zendesk admin page and look out for Channel Integrations button at the side bar.
  2. Select our “Telegram Channel” app.
  3. Select the “Accounts” tab.
  4. Select the “Add” button in order to configure the Telegram Bot token you copied before.
  5. Set an Integration Name to organize your accounts.
  6. Paste your Telegram Bot Token into Telegram Token field.
  7. Select the “Submit” button.

The new account is ready to start receiving your Telegram customers´ messages.
